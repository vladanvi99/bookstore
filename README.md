# Bookstore App

> This is Bookstore App.

Here you can add, remove your books, and track your progress with reading books.

## Setup

### Clone this repository

```bash
$ git clone https://github.com/vladanvi99/bookstore.git
$ cd bookstore
```

### Run project

```bash
$ npm install
$ npm start
```
